# 基于redux-thunk实现Todo案例

### 案例介绍

Todo是使用create-react-app脚手架并且基于Ant Design 构建的React案例

### 环境搭建

- node.js：v12.13.0
- react：v16.12.0
- IDE：VScode
- yarn：v1.19.2

### 运行启动

```shell
git clone https://github.com/q-linyu/react-todo-thunk.git

# 安装依赖
yarn

# 启动
yarn start

# 访问url
localhost:3000
```
